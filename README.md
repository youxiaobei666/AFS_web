# 宠友社项目

## 项目介绍

### 项目概述：

- **项目名称：** 宠友社
- **目的：** 救助全国的流浪猫和流浪狗
- 技术栈：
  - **前端：** Vue3 和 Element-plus
  - **后端：** Node.js Express
  - **数据库：** MySQL

后端 git 地址：https://github.com/youxiaobei666/AFS_back

测试用户：admin 

密码：666666

## 主要模块

前台页面 | 个人中心 | 消息中心 | 用户设置 | 权限管理 | 动物中心
